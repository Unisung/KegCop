About

KegCop is open source iDevice application that monitors a kegerator by having drinkers create user accounts associated with the kegerator. The admin of the kegerator will login with the root account, and then issue credits to the users of the kegerator. The source code for the application can be found here, https://github.com/ipatch/KegCop

Install

  1. Step1: Launch Cydia
  2. Step2: Click the "Manage" button.
  3. Step3: Click the "Sources" button.
  4. Step4: Click the "Edit" button.
  5. Step5: Click the "Add" button.
  6. Step6: Input the following URL"http://chrisrjones.com/repo"
  7. Step7: Click "Add Source" button.
  8. Step8: Select the repo, then select "KegCop".
  9. Step9: Click the "Install" button, then click "Confirm".
